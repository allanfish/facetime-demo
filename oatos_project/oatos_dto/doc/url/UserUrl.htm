同事REST服务

功能列表:

  • 添加企业用户
  • 用户导入
  • 获取单个用户信息
  • 修改用户信息
  • 批量修改用户信息
  • 修改员工信息和登录密码
  • 取某用户的在线状态
  • 取在线同事(基本信息)
  • 统计企业在线用户数
  • 按企业id取企业用户，包含角色信息
  • 删除企业员工
  • 获取所有同事的在线状态
  • 按企业id取企业用户，不包括状态及角色
  • 按企业id取企业部门和成员，不包括角色和状态
  • 按企业id取企业部门和成员，包含角色，不包含状态
  • 按用户id取企业部门和成员,包含在线状态，不包含角色信息
  • 取在线用户和离线用户ID
  • 修改用户头像

添加企业用户

请求地址:"/sc/ent/addEnterpriseUser"

参数:

  • EnterpriseUserDTO 企业员工DTO

正常返回:

  • userId 用户的ID

异常返回:

  • errorCheckHashkey数据未通过安全验证
  • errorEnterpriseNotExist企业不存在
  • errorUserNoOver用户数已经用完
  • errorPayExpired购买的用户已经过期
  • errorEmployeeAlreadyExist用户名已经存在
  • error500服务器内部错误

取某用户的在线状态

请求地址:"/sc/login/getUserStatus"

参数:

  • userId用户ID

正常返回:

  • UserStatus.Logout 退出
  • UserStatus.Busy 忙碌
  • UserStatus.Corbet隐身
  • UserStatus.Leave离开
  • UserStatus.Offline 不在线
  • UserStatus.Online 在线

异常返回:

  • error500服务器内部错误
