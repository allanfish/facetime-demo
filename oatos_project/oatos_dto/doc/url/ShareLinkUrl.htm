创建共享链接

请求地址:"/sc/shareLink/createShareLink"

参数:

  • ShareLinkDTO 共享链接DTO

正常返回:

  • ShareLinkDTO 共享链接DTO

异常返回:

  • error500 服务器内部错误

更新共享链接

请求地址:"/sc/shareLink/updateShareLink"

参数:

  • ShareLinkDTO 共享链接DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

删除共享链接

请求地址:"/sc/shareLink/deleteShareLink"

参数:

  • linkId 链接ID

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

获取共享链接

请求地址:"/pub/shareLink/getShareLinkByCode"

参数:

  • linkCode 链接代码

正常返回:

  • ShareLinkInfoDTO 共享链接信息DTO

异常返回:

  • error500 服务器内部错误

获取共享链接

请求地址:"/sc/shareLink/getShareLinkByEntId"

参数:

  • enterpriseId 企业ID

正常返回:

  • ShareLinksDTO 共享链接DTO

异常返回:

  • error500 服务器内部错误

获取共享链接

请求地址:"/sc/shareLink/getExistShareLink"

参数:

  • ShareLinkDTO 共享链接DTO

正常返回:

  • ShareLinkDTO 共享链接DTO

异常返回:

  • error500 服务器内部错误

获取共享链接

请求地址:"/sc/shareLink/getShareLinkByLinkId"

参数:

  • linkId

正常返回:

  • ShareLinkDTO 共享链接DTO

异常返回:

  • error500 服务器内部错误

验证共享链接是否可以下载

请求地址:"/sc/shareLink/checkShareLinkDownload"

参数:

  • linkID

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误
  • errorFileNotFound文件不存在
  • errorExpirationTimeOver过期了
  • errorDownCountOver超过下载数了
  • errorFlowOver流量超出

更新共享链接下载次数

请求地址:"/sc/shareLink/updateShareLinkDownCount"

参数:

  • linkId:size：外链id+文件大小kb

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

发送共享链接

请求地址:"/sc/shareLink/sendShareLinkMail"

参数:

  • ShareLinkMailDTO 外链邮件DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误
