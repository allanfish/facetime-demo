个人收藏夹REST服务

功能列表:

  • 添加个人收藏夹文件列表
  • 取消个人收藏夹文件列表
  • 获得个人收藏夹文件列表

添加个人收藏夹文件列表

请求地址:"/sc/favoriteFile/addFavoriteFile"

参数:

  • FavoriteFilesDTO 个人收藏文件夹DTO

正常返回:

  • CommConstants.OK OK

异常返回:

  • error500：系统错误

取消个人收藏夹文件列表

请求地址:"/sc/favoriteFile/delFavoriteFile"

参数:

  • FavoriteFilesDTO 个人收藏文件夹DTO

正常返回:

  • CommConstants.OK OK

异常返回:

  • error500：系统错误

获得个人收藏夹文件列表

请求地址:"/sc/favoriteFile/getFavoriteFile"

参数:

  • userId用户ID

正常返回:

  • ShareFilesDTO 企业网盘文件列表DTO

异常返回:

  • error500：系统错误
