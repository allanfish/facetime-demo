阻止联系人REST服务

功能列表:

  • 用户取阻止通信的用户列表(黑名单)
  • 用户设置阻止通信的用户列表(黑名单)
  • 取企业阻止通信列表(管理员设置不可见)
  • 管理员设置阻止通信列表(不可见)
  • 用户解除阻止通信的用户id
  • 管理员删除企业通信阻止(不可见)
  • 用户取被那些同事设置为阻止通信

用户取阻止通信的用户列表(黑名单)

请求地址:"/sc/block/getBlockListByUserId"

参数:

  • userId 用户ID

正常返回:

  • BlockDTO阻止人信息DTO

异常返回:

  • error500 服务器内部错误

用户设置阻止通信的用户列表(黑名单)

请求地址:"/sc/block/setBlockListByUserId"

参数:

  • BlockDTO 阻止人DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

取企业阻止通信列表(管理员设置不可见)

请求地址:"/sc/block/getEnterpriseBlocksByUserId"

参数:

  • userId 用户ID

正常返回:

  • BlockDTO 组织人DTO

异常返回:

  • error500 服务器内部错误

管理员设置阻止通信列表(不可见)

请求地址:"/sc/block/updateEnterpriseBlocks"

参数:

  • BlocksDTO阻止联系人列表DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

用户解除阻止通信的用户id

请求地址:"/sc/block/deleteBlock"

参数:

  • BlockDTO阻止人信息DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

管理员删除企业通信阻止(不可见)

请求地址:"/sc/block/deleteEnterpriseBlock"

参数:

  • BlockDTO阻止人信息DTO

正常返回:

  • CommConstants.OK_MARK OK

异常返回:

  • error500 服务器内部错误

用户取被那些同事设置为阻止通信

请求地址:"/sc/block/getBlockedByListByUserId"

参数:

  • userId 用户ID

正常返回:

  • BlockDTO阻止人信息DTO

异常返回:

  • error500 服务器内部错误
